┌──────────────────────────────────────────────────────────────────────────┐
│ TROUBLESHOOTING                                                          │
└──────────────────────────────────────────────────────────────────────────┘

PROBLEM: Terminal doesn't auto-start
FIX: Verify .vbs file in shell:startup folder, check Task Manager → Startup

PROBLEM: Crypto data shows "API ERROR"
FIX: Wait 60s and refresh, check internet connection

PROBLEM: TradFi shows "WAITING FOR KEY"
FIX: Add AlphaVantage API key in SYSTEM SETTINGS

PROBLEM: TradFi shows "API LIMIT"
FIX: Free tier = 5 calls/min. Wait 60s or upgrade to premium.

PROBLEM: Windows blocks .vbs file
FIX: Right-click file → Properties → Unblock → Apply → OK

┌──────────────────────────────────────────────────────────────────────────┐
│ ADVANCED OPTIONS                                                         │
└──────────────────────────────────────────────────────────────────────────┘

Open in specific browser:
  Edit .bat file, replace 'start' with:
  - Chrome:   start chrome "path\to\file.html"
  - Firefox:  start firefox "path\to\file.html"
  - Edge:     start msedge "path\to\file.html"

Fullscreen (Kiosk Mode):
  start chrome --kiosk "path\to\file.html"

Delay startup by 30 seconds:
  Add before 'start' command: timeout /t 30 /nobreak >nul

Clear all saved data:
  Open terminal → SYSTEM SETTINGS → WIPE DATA → Refresh
