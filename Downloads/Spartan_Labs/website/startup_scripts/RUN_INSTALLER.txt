OPTION 1: Windows File Explorer (EASIEST)
═══════════════════════════════════════════════════════════════

1. Press:  Win + E (to open File Explorer)

2. Navigate to:
   C:\Users\Quantum\Downloads\Spartan_Labs\website\startup_scripts

3. Find file:  INSTALL_FIXED_STARTUP_SCRIPTS.bat

4. Right-click it → Select "Run as Administrator"

5. Follow the prompts!

═══════════════════════════════════════════════════════════════

OPTION 2: PowerShell (From any directory)
═══════════════════════════════════════════════════════════════

Copy and paste this into PowerShell:

.\INSTALL_FIXED_STARTUP_SCRIPTS.bat

═══════════════════════════════════════════════════════════════

OPTION 3: PowerShell (Run directly with full path)
═══════════════════════════════════════════════════════════════

TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

Error: "Cannot find path"
→ Check if the folder exists at that location
→ Try navigating manually in File Explorer

Error: "Access denied"
→ Right-click the .bat file
→ Select "Run as Administrator"

Error: "Script is disabled"
→ The installer will fix this automatically
→ Just press Y when prompted

═══════════════════════════════════════════════════════════════

RECOMMENDED: Use Option 1 (Windows File Explorer)
It's the most reliable and gives you visual confirmation.
