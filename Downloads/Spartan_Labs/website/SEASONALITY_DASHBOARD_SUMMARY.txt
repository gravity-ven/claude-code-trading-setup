================================================================================
CALCULATIONS & METHODOLOGY
================================================================================

Monthly Return Calculation:
  return = ((close[month] - close[month-1]) / close[month-1]) * 100

Average Monthly Return:
  avg = sum(all_returns_for_month) / count(returns)

Win Rate:
  win_rate = (count(positive_returns) / count(total_returns)) * 100

Standard Deviation:
  stddev = sqrt(sum((return - avg)^2) / count)

Color Scaling:
  - Negative: Red gradient based on intensity
  - Positive: Green gradient based on intensity
  - Normalized across min/max values
