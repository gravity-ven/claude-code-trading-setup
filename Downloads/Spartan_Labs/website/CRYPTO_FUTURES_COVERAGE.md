## How Crypto Futures Analysis Works

### Step 1: CFTC Data Collection
Every Friday at 3:30 PM ET, CFTC publishes data showing:
- **Long positions**: Institutions betting Bitcoin will go UP
- **Short positions**: Institutions betting Bitcoin will go DOWN
- **Net positions**: Overall institutional bias

### Step 2: COT Index Calculation
```
Formula: ((Current_Net - Min_Net) / (Max_Net - Min_Net)) × 100

Example (Bitcoin):
  Current Net: +15,000 contracts (institutions are LONG)
  26-Week Low: -5,000 contracts
  26-Week High: +20,000 contracts

  COT Index = ((15,000 - (-5,000)) / (20,000 - (-5,000))) × 100
            = (20,000 / 25,000) × 100
            = 80.0

Interpretation:
  * 80/100 = Bullish (institutions heavily long)
  * When COT Index > 75, expect upward pressure
  * When COT Index < 25, expect downward pressure
```

### Step 3: Signal Generation
```
IF COT Index > 75:
    Signal: BUY Bitcoin
    Reason: Institutional accumulation

IF COT Index < 25:
    Signal: SELL Bitcoin
    Reason: Institutional distribution
```

## Comparison: Spot vs. Futures

### What COT Shows:
- **Futures data**: Institutional positioning (what this system analyzes)
- **NOT spot data**: Retail trading on Coinbase, etc.

### Why Futures Matter More:
1. **Bigger money**: Institutions trade futures (billions)
2. **Leverage**: Futures amplify moves (10x-20x leverage)
3. **Price leader**: Futures often move BEFORE spot

**Example**:
- Institutions buy Bitcoin futures heavily (COT shows this)
- Futures price rises first
- Arbitrage traders buy spot to match futures
- Spot price follows

**Your edge**: See institutional buying BEFORE spot catches up
